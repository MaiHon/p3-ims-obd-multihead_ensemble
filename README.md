Baseline-v2
===
## Usage
### Train
- `python main.py --config "your config"`

### Evaluation
- `python eval.py --eval_config "your eval config"`

### Inference
- If you want to inference all the test batches,
    - `python inference.py`

- x is Number of images to use in inference.
    - `python inference.py --limit x`
